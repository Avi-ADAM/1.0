# תכנית: סרטון קידום ל-1💗1 עם Remotion

מסמך זה מסביר **איך** ליצור סרטון קידום לאתר [1lev1.com](https://1lev1.com)
באמצעות [Remotion](https://remotion.dev) (יצירת וידאו בקוד React), וכולל פרויקט
מוכן להעתקה תחת התיקייה [`/remotion`](../remotion).

> TL;DR — הקבצים כבר כאן. `cd remotion && npm install && npm run dev` פותח את
> Remotion Studio. `npm run render` מפיק MP4. ערכו טקסט/צבעים בקבצי `src/scenes/*`.

---

## 1. למה Remotion (ולא עורך וידאו רגיל)

- **הכול בקוד וב-Git** — כל שינוי בתסריט/צבע/תזמון הוא קומיט, ניתן לשחזור ולעריכה.
- **מותג עקבי** — אנחנו שואבים את הצבעים והגופן (Rubik, גרדיאנט הזהב) ישירות
  מהאתר (`tailwind.config.cjs`, `app.postcss`) דרך `remotion/src/theme.ts`.
- **רינדור לכמה פורמטים** ממקור אחד: 16:9 לאתר/יוטיוב, 9:16 לרילז/סטוריז, 1:1 לפיד.
- **תוכן דינמי** — אפשר להזרים נתונים אמיתיים (מס׳ רקמות, צילומי מסך) לתוך הסרטון.

## 5. נכסים (Assets)

- **גופן** — Rubik נטען אוטומטית דרך `@remotion/google-fonts/Rubik` ב-`Root.tsx`.
- **לוגו** — כרגע מורכב מטקסט "1💗1" + אמוג׳י לב (זהה לאתר). להחלפה בלוגו תמונה:
  שימו קובץ ב-`remotion/public/logo.png` והשתמשו ב-`<Img src={staticFile('logo.png')} />`.
- **צילומי מסך אמיתיים** — מומלץ! צלמו 2–3 מסכים מהאתר (דשבורד רקמה, גאנט, הצבעות),
  שימו ב-`remotion/public/` והוסיפו אותם לסצנת התכונות להוכחה ויזואלית.
- **מוזיקה** — שימו `remotion/public/music.mp3` והדליקו `WITH_MUSIC = true` ב-`Promo.tsx`.
  בחרו רצועה ללא תמלוגים (למשל מ-Pixabay/YouTube Audio Library).

## 6. צעדים לביצוע

```bash
# 1. התקנה
cd remotion
npm install

# 2. עריכה חיה בסטודיו (סקראב בטיימליין, תצוגה מקדימה של כל הפורמטים)
npm run dev          # http://localhost:3000

# 3. רינדור ל-MP4
npm run render           # 1920x1080  -> out/promo-landscape.mp4
npm run render:vertical  # 1080x1920  -> out/promo-vertical.mp4
npm run render:square    # 1080x1080  -> out/promo-square.mp4
```

דרישות: Node 18+. ברינדור הראשון Remotion מתקין Chromium אוטומטית.

## 8. שיפורים עתידיים (Nice-to-have)

- **גרסה רב-לשונית** — להוציא מחרוזות לקובץ `i18n.ts` ולרנדר עברית/אנגלית מאותו קוד.
- **נתונים חיים** — להזין `inputProps` (מס׳ רקמות פעילות, חברים) דרך
  `--props` ל-`remotion render`, ולהציג מספרים אמיתיים עם `@number-flow`.
- **CI render** — GitHub Action שמרנדר את הסרטון בכל שינוי ב-`remotion/`
  ומעלה אותו כ-artifact.

המבנה המלא של הפרויקט מתועד ב-[`/remotion/README.md`](../remotion/README.md).
